- # Data Structures
HC是编制管控体系的意思。编制管控是在一个经营财年下，编制变化执行的管理。整个财年上编制管控分为：编制计划定制、编制计划执行、编制执行结束、编制执行总结等4个环节。
  - HC核心数据对象、HC基础信息、HC台账
    - HC基础信息：编制ID、HC说明、HC创建日期、HC状态（空编、占编、在编、锁编、撤编）、HC所属组织编号、HC所属岗位编号、HC所属主体编号
    - HC台账信息：编制台账ID、父编制台账ID、编制ID、台账类型（入职、离职、调动、转正、转换、续签、换签、测评、盘点、绩效、晋升、降级、组织调整）、台账变更字段、变更字段变更前数值、变更字段变更后数值、台账执行状态（待执行、执行中、已执行、阻塞中、已撤销）

- # Functions

  - HC台账信息：CRUD功能
